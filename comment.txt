변수 = 익명함수;

* 자바스크립트 함수는 자료형이기 때문에 변수에 넣어서 사용

익명함수 형식:	function () {}
		() => {}

* 화살표 함수에서
- 하나의 인수를 전달할 경우 소괄호 생략가능
- 하나의 표현식을 리턴할 경우 중괄호 생략가능

ajax의 특징은 
페이지 새로 고침 없이
새 데이터를 반영할 수 있다는 것이다

외부에 있는 새 데이터를 계속적으로 반영하기 위해
순서 없이 비동기적으로 움직이는 함수를 만든다(async)

그러나 새 데이터를 반영하는 과정은
순서가 있어야 하므로 async 함수 내에서 await를 사용한다

예를들면

데이터 가져오기 -> 데이터를 상태에 저장
하는 순서가 필요할 것이다

모든 데이터를 가져올 때까지 기다리지 않는다면(await)
일부 데이터만 반영되어 혼란이 있을 수 있다
